N-BODY PROBLEM

- How do n objects move if each one graviationally attracts the others?
Examples:

Sun + Earth → 2-body problem

Sun + Earth + Moon → 3-body problem

solar system → N-body

The 2-body problem has an exact solution (elliptical orbits).

But for N ≥ 3, there is no general analytic solution.
Instead we approximate the motion with numerical methods.
